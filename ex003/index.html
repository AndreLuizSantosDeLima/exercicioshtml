Símbolos e emoji

Para usar simbolos como < e > é preciso colocar o &lt e &gt; desta forma os simbolos de > < apareceram,
se deseja outro simbolos so subistitua algumas letras que terá resultado diferentes, para saber como fazer
os símbolos confira a tabela UTF-8.

Alguns exemplos de simbolos são:
© = &copy;
® = &reg;
£ = &pound;
¢ = &cent;
¥ = &yen;
€ = &euro;
™ = &trade;
∞ = &infin;

Vamos adicionar alguns emoji:
😪 = &#x1F62A;
🖖 = &#x1F596;
🤓 = &#x1F913;
